# Terraform Learning Journey (AWS)

Working through Terraform from scratch, targeting AWS infrastructure.
All exercises use real AWS resources — every phase has its own folder.

## Structure
| Phase | Topic |
|-------|-------|
| 1 | Core concepts — providers, resources, state |
| 2 | Variables, outputs, data sources |
| 3 | Modules and reusability |
| 4 | Remote state and workspaces |
| 5 | CI/CD with GitHub Actions |
| 6 | Advanced patterns |

## Cost policy
All exercises are designed to be near-zero cost.
Resources are destroyed after each exercise.
